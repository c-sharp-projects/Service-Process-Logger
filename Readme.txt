Service Process Information Logger


Project Name : Service Process Information Logger.

Description : This project is used for loging (Storing        information in a file ) the Information about the current running processes.
  The Information is stored in every 1 hour .
For each day one new file is created and the name of file will be current date which if preceded by ProcessLog for eg ( ProcessLog_10_11_2018 ).
